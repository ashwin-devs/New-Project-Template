# 🚀 Android Project Template

A production-ready Android starter template built with **Clean Architecture** and modern Android development tools to help you kickstart scalable and maintainable applications.

---

## ✨ Features

- 🧱 Clean Architecture (Presentation, Domain, Data)
- 💉 Dependency Injection using Hilt
- 🌐 Networking with Retrofit
- 🗄️ Local Database with Room
- 🔄 Kotlin Coroutines & Flow
- 🧩 MVVM Architecture
- 📦 Modular & Scalable Code Structure
- 🖼️ Base Activity, Fragment & ViewModel setup
- ⚡ Ready-to-use boilerplate code

---

## 🛠️ Tech Stack

- **Kotlin**
- **Android Jetpack**
  - ViewModel
  - LiveData / Flow
  - Navigation Component
- **Hilt**
- **Retrofit**
- **Room Database**
- **Coroutines**
